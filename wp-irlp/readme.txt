This plugin has a set of tools for IRLP repeaters. Right now, it just has a widget that displays an IRLP repeater's status.

== Description ==

== Installation ==

1. Upload the `WP-IRLP` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
